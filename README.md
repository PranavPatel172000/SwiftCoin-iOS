# 📈 SwiftCoin - Crypto Tracker App (SwiftUI + Combine)

SwiftCoin is a modern iOS application built using **SwiftUI** and **Combine** that displays real-time cryptocurrency data.
The app fetches data from a public API and presents the **top cryptocurrencies**, including **top performers**, in a clean and responsive user interface.

---

## 🚀 Features

* 📊 View top 100 cryptocurrencies
* 📈 Highlight top-performing coins
* 🔄 Real-time data fetching from API
* ⚡ Reactive programming using Combine
* 🎨 Built with SwiftUI for modern UI
* 📱 Smooth and responsive user experience

---

## 🛠️ Tech Stack

* **SwiftUI** – Declarative UI framework
* **Combine** – Reactive programming & API handling
* **URLSession** – Network layer
* **MVVM Architecture** – Clean separation of concerns

---

## 🧠 Architecture

The app follows the **MVVM (Model-View-ViewModel)** pattern:

* **Model** → Represents crypto data from API
* **ViewModel** → Handles API calls using Combine and processes data
* **View** → SwiftUI views rendering UI based on state

---

## 🔗 API Integration

* Fetches cryptocurrency data from a public API
* Uses Combine for:

  * Data binding
  * Asynchronous network calls
  * Handling API responses
